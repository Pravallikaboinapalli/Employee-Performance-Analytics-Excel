# Employee Performance Analytics Dashboard

## Project Overview
This project is an interactive Employee Performance Analytics Dashboard created in Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Conditional Formatting

## Dashboard Features
- Total Employees
- Average Salary
- Attendance Analysis
- Performance Rating
- Department-wise Analysis
- Experience-wise Salary
- Gender Distribution
- Salary Band Analysis
- Interactive Filters

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- Data Visualization
- Excel Reporting
